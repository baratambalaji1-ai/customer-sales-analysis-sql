# Customer Sales Analysis & Segmentation (SQL)

## 📌 Project Overview
This project analyzes customer purchasing behavior using SQL. 
The goal is to calculate customer lifetime value, perform segmentation, 
and identify high-value and inactive customers.

---

## 🎯 Objectives
- Calculate Customer Lifetime Value (CLV)
- Perform customer segmentation based on spending
- Identify high-value customers
- Find customers who never placed orders
- Analyze order frequency and spending trends

---

## 🛠 SQL Concepts Used
- INNER JOIN
- LEFT JOIN
- GROUP BY
- HAVING
- CASE Statements
- Aggregations (SUM, COUNT, MAX)
- ORDER BY & LIMIT

---

## 📊 Key Insights
- Identified the highest spending customer
- Segmented customers into High, Medium, and Low value groups
- Calculated total spending and order frequency per customer
- Detected inactive customers using LEFT JOIN logic
